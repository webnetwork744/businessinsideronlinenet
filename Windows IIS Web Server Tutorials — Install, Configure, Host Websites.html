Windows IIS Web Server Tutorials

Learn to install, configure, secure, and troubleshoot IIS for hosting websites and applications on Windows.

1. Install IIS on Windows

  1. Press Win + R, type optionalfeatures → Enter.
  2. In Windows Features, check Internet Information Services.
  3. Select Web Management Tools and World Wide Web Services.
  4. Click OK, then open http://localhost to see IIS welcome page.

Keywords: install IIS Windows 10, install IIS Windows 11

2. IIS Manager Overview

Open IIS Manager with inetmgr. Left panel shows sites and app pools, middle panel shows features, right panel gives actions (start, stop, restart).

Keywords: IIS Manager tutorial, IIS configuration basics

3. Host a Website in IIS

  1. In IIS Manager, right-click Sites → Add Website.
  2. Enter site name, choose physical path (e.g., C:\inetpub\wwwroot\mysite).
  3. Set binding: http, port 80.
  4. Click OK → Test on http://localhost.

Keywords: add website IIS, host site IIS Windows

4. IIS Application Pools

Application Pools isolate apps for stability and security. Each pool runs with its own worker process.

  • Set .NET CLR version per site.
  • Recycle pools to refresh memory.

Keywords: IIS application pools explained, configure IIS app pool

5. Enable HTTPS (SSL Certificate)

  1. Create a self-signed certificate in IIS or install a trusted SSL cert.
  2. IIS Manager → Site → Bindings → Add HTTPS (port 443).
  3. Select certificate → Save.

Keywords: enable SSL IIS, HTTPS IIS setup

6. Deploy ASP.NET Website

  1. Publish project from Visual Studio to a folder.
  2. Create IIS site → point to published folder.
  3. Set App Pool to correct .NET CLR version.

Keywords: deploy ASP.NET IIS, host ASP.NET site IIS

7. Deploy PHP Website

  1. Install PHP for Windows (PHP Manager for IIS).
  2. Configure Handler Mappings in IIS for PHP.
  3. Point IIS site to PHP files.
  4. Test with phpinfo.php.

Keywords: deploy PHP IIS, run PHP on Windows IIS

8. Useful IIS Commands (PowerShell)

  • Restart IIS: iisreset
  • Start IIS: Start-Service W3SVC
  • Stop IIS: Stop-Service W3SVC

Keywords: IIS PowerShell commands, restart IIS Windows

9. Troubleshooting IIS

  • Check ports if site won’t load (80/443).
  • Give folder permissions to IUSR or IIS_IUSRS.
  • Enable detailed errors for 500 issues.

Keywords: troubleshoot IIS, fix IIS errors
