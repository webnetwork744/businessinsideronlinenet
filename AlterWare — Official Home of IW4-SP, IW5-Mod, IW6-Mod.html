Quick Start — Install & Join

  1. Obtain a legitimate copy of the Call of Duty title required by the mod (check IW4/IW5/IW6 compatibility).
  2. Visit the official AlterWare site and download the appropriate mod client.
  3. Follow the install guide and apply any required patches.
  4. Use the server browser in the client or community lists to find active servers.
  5. Join community Discords for server recommendations and support.

FAQ

Are AlterWare projects official?

No. AlterWare projects are community-driven and are not affiliated with the original game publishers. Always follow the game EULA and local laws.

Do I need the original game?

Yes — most mods require a legitimate copy of the matching Call of Duty title. See install instructions on the official site.

Where are active servers listed?

Active servers are shown in the client server browser and community lists on the AlterWare site and related Discord channels.
